reflexion-semana3.txtEs importante que los proyectos de software mantengan un registro de cambios porque permite organizar mejor el trabajo realizado. Herramientas como Git ayudan a guardar cada modificación hecha en los archivos del proyecto. Esto facilita corregir errores y recuperar versiones anteriores cuando sea necesario. Además, permite que varias personas trabajen en el mismo proyecto sin perder información. Git también mejora la seguridad y el control de los cambios realizados. Mantener una trazabilidad ayuda a identificar quién hizo cada modificación y cuándo se realizó. Esto hace que el trabajo sea más ordenado y profesional. Por eso, Git es una herramienta fundamental en el desarrollo de software moderno.
